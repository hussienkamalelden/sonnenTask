# Project Name: `Battery Monitor`

https://ibb.co/hJKvwzv8

# Project Description:

This project is a simple battery monitor that displays the battery's charging level over the last 24 hours.

# Project Stack:

- React
- TypeScript
- Tailwind CSS
- Apexcharts
- Vite

# Project Setup

Node version: v20.18.2

vite version: 6.1.0

to run the project:

```bash
npm install
npm run dev
```
